## AI i modele językowe

<div class="glossary-list">

<div class="glossary-term" id="ai">

### AI (Artificial Intelligence)

Sztuczna inteligencja. Systemy komputerowe naśladujące ludzkie myślenie: rozumienie tekstu, generowanie odpowiedzi, analiza danych. Claude to przykład AI.

<div class="glossary-related">Powiązane: <a href="#llm">LLM</a>, <a href="#prompt">Prompt</a>, <a href="#token">Token</a></div>
</div>

<div class="glossary-term" id="llm">

### LLM (Large Language Model)

Duży model językowy. Typ AI wyszkolony na ogromnych ilościach tekstu, który potrafi rozumieć i generować język naturalny. ChatGPT, Claude, Gemini — to wszystko LLM-y.

<div class="glossary-related">Powiązane: <a href="#ai">AI</a>, <a href="#geo">GEO</a>, <a href="#token">Token</a></div>
</div>

<div class="glossary-term" id="geo">

### GEO (Generative Engine Optimization)

Optymalizacja pod wyszukiwarki AI. Nowa dyscyplina polegająca na tym, żeby Twoje treści pojawiały się w odpowiedziach ChatGPT, Perplexity czy Google AI Overviews. Rozszerzenie tradycyjnego SEO na erę AI.

<div class="glossary-related">Powiązane: <a href="#seo">SEO</a>, <a href="#rag">RAG</a>, <a href="#llm">LLM</a></div>
</div>

<div class="glossary-term" id="rag">

### RAG (Retrieval-Augmented Generation)

Generowanie wspomagane wyszukiwaniem. Technika, w której AI najpierw wyszukuje istotne informacje (np. z Twojej strony), a dopiero potem generuje odpowiedź. Dzięki temu odpowiedzi są precyzyjniejsze i aktualne.

<div class="glossary-related">Powiązane: <a href="#geo">GEO</a>, <a href="#llm">LLM</a></div>
</div>

<div class="glossary-term" id="prompt">

### Prompt

Zapytanie lub instrukcja, którą podajesz AI. Np. "napisz artykuł o email marketingu" to prompt. Im lepszy prompt, tym lepszy wynik.

<div class="glossary-related">Powiązane: <a href="#ai">AI</a>, <a href="#llm">LLM</a></div>
</div>

<div class="glossary-term" id="token">

### Token

Jednostka tekstu dla AI. Mniej więcej 1 token = 0,75 słowa. Modele AI mają limit tokenów (okno kontekstu), dlatego llms.txt pomaga im efektywnie przetwarzać strony.

<div class="glossary-related">Powiązane: <a href="#llm">LLM</a>, <a href="#ai">AI</a></div>
</div>

</div>

## SEO — podstawy

<div class="glossary-list">

<div class="glossary-term" id="seo">

### SEO (Search Engine Optimization)

Optymalizacja pod wyszukiwarki. Zbiór praktyk sprawiających, że Twoja strona pojawia się wyżej w Google i innych wyszukiwarkach.

<div class="glossary-related">Powiązane: <a href="#geo">GEO</a>, <a href="#serp">SERP</a>, <a href="#keywords">Keywords</a></div>
</div>

<div class="glossary-term" id="serp">

### SERP (Search Engine Results Page)

Strona wyników wyszukiwania. To co widzisz po wpisaniu czegoś w Google — lista linków, snippety, reklamy.

<div class="glossary-related">Powiązane: <a href="#seo">SEO</a>, <a href="#featured-snippet">Featured snippet</a></div>
</div>

<div class="glossary-term" id="keywords">

### Keywords (słowa kluczowe)

Frazy, które ludzie wpisują w wyszukiwarkę. Np. "najlepszy CRM dla małych firm". SEO polega m.in. na optymalizacji treści pod te frazy.

<div class="glossary-related">Powiązane: <a href="#long-tail-keywords">Long-tail</a>, <a href="#lsi">LSI</a>, <a href="#keyword-difficulty">Keyword difficulty</a>, <a href="#search-intent">Search intent</a></div>
</div>

<div class="glossary-term" id="long-tail-keywords">

### Long-tail keywords

Długie frazy kluczowe. Np. zamiast "CRM" to "najlepszy darmowy CRM dla jednoosobowej działalności". Mniej wyszukiwań, ale łatwiej się pozycjonować i wyższa konwersja.

<div class="glossary-related">Powiązane: <a href="#keywords">Keywords</a>, <a href="#search-intent">Search intent</a></div>
</div>

<div class="glossary-term" id="lsi">

### LSI (Latent Semantic Indexing)

Semantycznie powiązane terminy. Google rozumie kontekst — jeśli piszesz o "kawie", oczekuje też słów jak "espresso", "parzenie", "ziarna". Plugin automatycznie je identyfikuje.

<div class="glossary-related">Powiązane: <a href="#keywords">Keywords</a>, <a href="#seo">SEO</a></div>
</div>

<div class="glossary-term" id="keyword-difficulty">

### Keyword difficulty

Trudność słowa kluczowego. Skala 0-100 pokazująca jak trudno jest zdobyć wysoką pozycję na daną frazę. Im wyżej, tym trudniej.

<div class="glossary-related">Powiązane: <a href="#keywords">Keywords</a>, <a href="#domain-authority">Domain Authority</a></div>
</div>

<div class="glossary-term" id="search-intent">

### Search intent (intencja wyszukiwania)

Czego szuka użytkownik: informacji (informacyjna), konkretnej strony (nawigacyjna), porównania produktów (komercyjna), zakupu (transakcyjna).

<div class="glossary-related">Powiązane: <a href="#keywords">Keywords</a>, <a href="#long-tail-keywords">Long-tail</a></div>
</div>

</div>

## SEO — techniczne

<div class="glossary-list">

<div class="glossary-term" id="on-page-seo">

### On-page SEO

Optymalizacja na stronie. Wszystko co możesz zmienić bezpośrednio na swojej stronie: tytuły, nagłówki, treści, meta tagi, linki wewnętrzne.

<div class="glossary-related">Powiązane: <a href="#meta-tags">Meta tags</a>, <a href="#schema-markup">Schema markup</a></div>
</div>

<div class="glossary-term" id="schema-markup">

### Schema markup / Structured data

Dane strukturalne. Specjalny kod (JSON-LD) dodawany do strony, który mówi Google i AI czym jest treść — artykuł, produkt, FAQ, firma lokalna itp. Pomaga pojawiać się w rich snippets.

<div class="glossary-related">Powiązane: <a href="#json">JSON</a>, <a href="#meta-tags">Meta tags</a>, <a href="#featured-snippet">Featured snippet</a></div>
</div>

<div class="glossary-term" id="core-web-vitals">

### Core Web Vitals

Metryki wydajności strony mierzone przez Google: LCP (jak szybko ładuje się główna treść), INP (jak szybko strona reaguje na kliknięcia), CLS (czy elementy nie przeskakują podczas ładowania).

</div>

<div class="glossary-term" id="meta-tags">

### Meta tags

Ukryte tagi HTML opisujące stronę. Najważniejsze: meta title (tytuł w Google), meta description (opis pod tytułem). Nie widać ich na stronie, ale widzisz je w wynikach wyszukiwania.

<div class="glossary-related">Powiązane: <a href="#on-page-seo">On-page SEO</a>, <a href="#schema-markup">Schema markup</a></div>
</div>

<div class="glossary-term" id="sitemap">

### Sitemap

Mapa strony w formacie XML. Lista wszystkich URL-i na Twojej stronie, którą podajesz Google żeby wiedział co zaindeksować.

<div class="glossary-related">Powiązane: <a href="#robots-txt">robots.txt</a></div>
</div>

<div class="glossary-term" id="robots-txt">

### robots.txt

Plik tekstowy mówiący wyszukiwarkom i crawlerom AI, które części strony mogą odwiedzać, a które nie.

<div class="glossary-related">Powiązane: <a href="#sitemap">Sitemap</a></div>
</div>

<div class="glossary-term" id="canonical-url">

### Canonical URL

Adres kanoniczny. Gdy masz kilka stron z podobną treścią, canonical mówi Google "ta wersja jest oficjalna".

</div>

<div class="glossary-term" id="eeat">

### E-E-A-T

Experience, Expertise, Authoritativeness, Trustworthiness. Framework Google do oceny jakości treści. Chodzi o to, żeby treść była pisana przez kogoś z doświadczeniem i wiedzą, na wiarygodnej stronie.

<div class="glossary-related">Powiązane: <a href="#seo">SEO</a>, <a href="#content-brief">Content brief</a></div>
</div>

</div>

## Linkowanie

<div class="glossary-list">

<div class="glossary-term" id="backlinks">

### Backlinks

Linki z innych stron prowadzące do Twojej. Im więcej wartościowych backlinków, tym wyższa pozycja. To jak rekomendacje od innych stron.

<div class="glossary-related">Powiązane: <a href="#domain-authority">Domain Authority</a>, <a href="#anchor-text">Anchor text</a></div>
</div>

<div class="glossary-term" id="internal-links">

### Internal links (linki wewnętrzne)

Linki między stronami w ramach Twojej witryny. Pomagają Google zrozumieć strukturę i przekazują "moc SEO" między stronami.

<div class="glossary-related">Powiązane: <a href="#pillar-page">Pillar page</a>, <a href="#anchor-text">Anchor text</a></div>
</div>

<div class="glossary-term" id="domain-authority">

### Domain Authority (DA)

Autorytet domeny. Metryka (0-100) pokazująca siłę SEO całej domeny. Im wyższy, tym łatwiej rankujesz na nowe frazy.

<div class="glossary-related">Powiązane: <a href="#backlinks">Backlinks</a>, <a href="#keyword-difficulty">Keyword difficulty</a></div>
</div>

<div class="glossary-term" id="anchor-text">

### Anchor text

Tekst klikalnego linka. Zamiast "kliknij tutaj" lepiej napisać "najlepsze narzędzia email marketing" — mówi Google o czym jest strona docelowa.

<div class="glossary-related">Powiązane: <a href="#backlinks">Backlinks</a>, <a href="#internal-links">Internal links</a></div>
</div>

</div>

## Treści i content marketing

<div class="glossary-list">

<div class="glossary-term" id="content-brief">

### Content brief

Brief contentowy. Dokument opisujący copywriterowi co i jak napisać: docelowe keywords, strukturę artykułu, długość, konkurencję do prześcignięcia.

<div class="glossary-related">Powiązane: <a href="#keywords">Keywords</a>, <a href="#pillar-page">Pillar page</a></div>
</div>

<div class="glossary-term" id="pillar-page">

### Pillar page

Strona filarowa. Długi, kompleksowy artykuł (3 000-5 000 słów) na główny temat, do którego linkują mniejsze artykuły klastrowe.

<div class="glossary-related">Powiązane: <a href="#topic-cluster">Topic cluster</a>, <a href="#internal-links">Internal links</a></div>
</div>

<div class="glossary-term" id="topic-cluster">

### Topic cluster

Klaster tematyczny. Grupa powiązanych artykułów linkujących do pillar page i między sobą. Np. pillar: "email marketing" to klastry: "automatyzacja emaili", "segmentacja listy", "A/B testy maili".

<div class="glossary-related">Powiązane: <a href="#pillar-page">Pillar page</a>, <a href="#internal-links">Internal links</a></div>
</div>

<div class="glossary-term" id="thin-content">

### Thin content

Cienka treść. Strony z bardzo małą ilością wartościowej treści. Google je penalizuje.

<div class="glossary-related">Powiązane: <a href="#eeat">E-E-A-T</a>, <a href="#content-brief">Content brief</a></div>
</div>

<div class="glossary-term" id="featured-snippet">

### Featured snippet

Wyróżniony fragment. Blok tekstu wyświetlany na samej górze Google (pozycja 0). Plugin pomaga optymalizować treść, żeby go zdobyć.

<div class="glossary-related">Powiązane: <a href="#serp">SERP</a>, <a href="#schema-markup">Schema markup</a></div>
</div>

</div>

## Lokalne SEO

<div class="glossary-list">

<div class="glossary-term" id="gbp">

### GBP (Google Business Profile)

Profil firmy w Google. Wizytówka Twojej firmy pojawiająca się w Google Maps i w wynikach lokalnych. Dawniej Google My Business.

<div class="glossary-related">Powiązane: <a href="#nap">NAP</a>, <a href="#local-pack">Local pack</a></div>
</div>

<div class="glossary-term" id="nap">

### NAP

Name, Address, Phone. Trzy podstawowe dane firmy, które muszą być identyczne wszędzie w internecie — na stronie, w katalogach, na mapach.

<div class="glossary-related">Powiązane: <a href="#gbp">GBP</a>, <a href="#citation">Citation</a></div>
</div>

<div class="glossary-term" id="citation">

### Citation (cytacja)

Wzmianka o Twojej firmie w internecie zawierająca NAP. Katalogi jak Yelp, Facebook, branżowe katalogi — każdy to osobna cytacja.

<div class="glossary-related">Powiązane: <a href="#nap">NAP</a>, <a href="#gbp">GBP</a></div>
</div>

<div class="glossary-term" id="local-pack">

### Local pack

Pakiet lokalny. Blok 3 firm z mapą wyświetlany w Google przy wyszukiwaniach lokalnych (np. "pizzeria Warszawa").

<div class="glossary-related">Powiązane: <a href="#gbp">GBP</a>, <a href="#seo">SEO</a></div>
</div>

</div>

## Technologia i konfiguracja

<div class="glossary-list">

<div class="glossary-term" id="mcp">

### MCP (Model Context Protocol)

Protokół komunikacji między narzędziami AI a zewnętrznymi serwisami. Dzięki MCP plugin SEO Machine może pobierać dane z Ahrefs czy Similarweb. Pomyśl o tym jak o "wtyczce do wtyczki".

<div class="glossary-related">Powiązane: <a href="#api">API</a>, <a href="#plugin">Plugin</a></div>
</div>

<div class="glossary-term" id="api">

### API (Application Programming Interface)

Interfejs programistyczny. Sposób komunikacji między programami. Gdy SEO Machine pobiera dane z Ahrefs — robi to przez API Ahrefs.

<div class="glossary-related">Powiązane: <a href="#rest-api">REST API</a>, <a href="#json">JSON</a>, <a href="#mcp">MCP</a></div>
</div>

<div class="glossary-term" id="rest-api">

### REST API

Najpopularniejszy styl API. Używa standardowych adresów URL do wysyłania i odbierania danych. Np. `GET https://api.ahrefs.com/keywords` — pobiera dane keywords.

<div class="glossary-related">Powiązane: <a href="#api">API</a>, <a href="#json">JSON</a></div>
</div>

<div class="glossary-term" id="json">

### JSON (JavaScript Object Notation)

Format danych czytelny dla ludzi i maszyn. Wygląda tak: `{"name": "SEO Machine", "version": "2.0.0"}`. Używany w konfiguracjach MCP, schema markup i odpowiedziach API.

<div class="glossary-related">Powiązane: <a href="#api">API</a>, <a href="#schema-markup">Schema markup</a></div>
</div>

<div class="glossary-term" id="env">

### Plik .env / ENV variables

Zmienne środowiskowe. Plik `.env` przechowuje poufne dane (klucze API, hasła) poza kodem. Zamiast wpisywać klucz Ahrefs w kodzie, trzymasz go w `.env` i odwołujesz się jako `AHREFS_API_KEY`.

</div>

<div class="glossary-term" id="cli">

### CLI (Command Line Interface)

Interfejs wiersza poleceń. Terminal/konsola, w której wpisujesz komendy tekstowe zamiast klikać. Claude Code CLI to wersja Claude działająca w terminalu.

<div class="glossary-related">Powiązane: <a href="#ide">IDE</a>, <a href="#plugin">Plugin</a></div>
</div>

<div class="glossary-term" id="plugin">

### Plugin

Dodatek rozszerzający funkcjonalność programu. SEO Machine to plugin do Claude — dodaje 16 komend SEO, których Claude domyślnie nie ma.

<div class="glossary-related">Powiązane: <a href="#mcp">MCP</a>, <a href="#ide">IDE</a></div>
</div>

<div class="glossary-term" id="ide">

### IDE (Integrated Development Environment)

Zintegrowane środowisko programistyczne. Edytor kodu z dodatkowymi narzędziami. Cursor, VS Code, Windsurf — to IDE, w których możesz zainstalować SEO Machine.

<div class="glossary-related">Powiązane: <a href="#cli">CLI</a>, <a href="#plugin">Plugin</a></div>
</div>

</div>

## Metryki i scoring

<div class="glossary-list">

<div class="glossary-term" id="ctr">

### CTR (Click-Through Rate)

Współczynnik klikalności. Procent osób, które kliknęły Twój link po zobaczeniu go w wynikach. CTR 5% = na 100 wyświetleń, 5 kliknięć.

<div class="glossary-related">Powiązane: <a href="#serp">SERP</a>, <a href="#seo">SEO</a></div>
</div>

<div class="glossary-term" id="flesch-reading-ease">

### Flesch Reading Ease

Wskaźnik czytelności tekstu (0-100). 60-70 to optymalny zakres dla treści SEO — zrozumiały dla szerokiej publiczności, ale nie infantylny.

</div>

<div class="glossary-term" id="bounce-rate">

### Bounce rate

Współczynnik odrzuceń. Procent osób, które odwiedziły stronę i wyszły bez żadnej interakcji. Niższy = lepszy.

<div class="glossary-related">Powiązane: <a href="#ctr">CTR</a>, <a href="#conversion-rate">Conversion rate</a></div>
</div>

<div class="glossary-term" id="conversion-rate">

### Conversion rate

Współczynnik konwersji. Procent odwiedzających, którzy wykonali pożądaną akcję (zakup, zapis na newsletter, pobranie pliku).

<div class="glossary-related">Powiązane: <a href="#bounce-rate">Bounce rate</a>, <a href="#ctr">CTR</a></div>
</div>

</div>
