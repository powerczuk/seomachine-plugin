Ten przewodnik przeprowadzi Cię przez kompletny workflow tworzenia treści — od badania słów kluczowych do gotowego artykułu — w około 5 minut.

## Krok 1 — Zbadaj temat

```
/research "najlepsze narzędzia email marketing dla małych firm"
```

Otrzymasz:

- 15–25 okazji keywords z wolumenem, difficulty i intencją
- Analizę krajobrazu SERP (co rankuje, formaty treści, długości)
- Luki konkurencji (czego brakuje w topowych wynikach)
- 3 propozycje meta title i description

## Krok 2 — Wygeneruj brief contentowy

```
/content-brief "najlepsze narzędzia email marketing dla małych firm"
```

Otrzymasz brief gotowy dla copywritera:

- Rekomendowaną strukturę artykułu (hierarchia H1, H2, H3)
- Keywords do uwzględnienia (primary, secondary, pytania, LSI)
- Strategię featured snippet
- Rekomendacje schema markup

## Krok 3 — Napisz artykuł

```
/write-seo "najlepsze narzędzia email marketing dla małych firm" 2500
```

Produkuje pełny artykuł SEO o ~2 500 słów:

- Wstęp z hookiem (metoda APP)
- Sekcje poparte danymi i dowodami
- Mini-stories zwiększające zaangażowanie
- Kontekstowe CTA
- Automatyczny scoring jakości (0–100)

## Krok 4 — Sprawdź wynik

Artykuł jest automatycznie oceniany po napisaniu. Jeśli potrzebuje poprawek:

```
/optimize [plik artykułu]
```

Zastosuje poprawki i pokaże porównanie przed/po. Cel: 70+ punktów do publikacji.

## Krok 5 — Dodaj meta tagi i schema

```
/meta-tags [plik artykułu]
/schema-gen [URL artykułu] Article FAQ
```
