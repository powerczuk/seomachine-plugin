## Czym jest SEO Machine?

SEO Machine to plugin do narzędzi opartych na Claude, który zamienia Claude w kompletną stację roboczą SEO. Udostępnia 16 slash komend, 5 modułów wiedzy (skills) i 4 autonomicznych agentów obejmujących cały cykl SEO — od badania słów kluczowych przez tworzenie treści, optymalizację, audyty techniczne po widoczność w wyszukiwarkach AI.

## Komendy w skrócie

### Tworzenie i optymalizacja treści

| Komenda | Co robi |
|---------|---------|
| `/research` | Badanie słów kluczowych, analiza SERP, identyfikacja luk konkurencji |
| `/write-seo` | Pisanie zoptymalizowanych artykułów SEO (2 000–3 000+ słów) |
| `/optimize` | Analiza i optymalizacja istniejącej treści ze scoringiem 0–100 |
| `/analyze` | Głęboka analiza jakości SEO z wielomodułowym scoringiem |
| `/content-brief` | Generowanie briefów contentowych gotowych dla copywriterów |
| `/meta-tags` | Generowanie zoptymalizowanych meta title, description i OG tags |
| `/rewrite` | Odświeżanie spadających treści w celu odzyskania pozycji |

### Architektura strony i techniczne SEO

| Komenda | Co robi |
|---------|---------|
| `/audit` | Pełny audyt SEO — techniczny, on-page, content, konkurencja |
| `/topic-clusters` | Mapowanie klastrów tematycznych z pillar pages i linkowaniem wewnętrznym |
| `/internal-links` | Strategiczna analiza i rekomendacje linkowania wewnętrznego |
| `/schema-gen` | Generowanie JSON-LD schema markup dla dowolnego typu strony |
| `/low-hanging-fruit` | Znajdowanie szybkich wygranych — keywords i strony do natychmiastowego wzrostu |

### GEO i wyszukiwanie AI

| Komenda | Co robi |
|---------|---------|
| `/geo-audit` | Audyt widoczności w wyszukiwarkach AI (ChatGPT, Perplexity, Gemini) |
| `/llms-txt` | Generowanie pliku llms.txt do optymalizacji crawlerów AI |

### Lokalne SEO

| Komenda | Co robi |
|---------|---------|
| `/local-seo` | Audyt lokalnego SEO — GBP, cytacje, NAP, lokalne keywords |

### Strategia

| Komenda | Co robi |
|---------|---------|
| `/competitors` | Głęboka analiza konkurencji z battlecards |

## Integracje

SEO Machine łączy się z zewnętrznymi narzędziami przez MCP (Model Context Protocol) zapewniając większą precyzję danych:

<CardGrid>
  <Card title="🔍 Ahrefs" icon="magnifier">
    Wolumeny słów kluczowych, difficulty scores, dane o backlinkach, pozycje w rankingu
  </Card>
  <Card title="📊 Similarweb" icon="bars">
    Szacunki ruchu, dane o odbiorcach, competitive intelligence
  </Card>
  <Card title="🔎 Google Search Console" icon="document">
    Twoje realne wyświetlenia, kliknięcia, CTR i pozycje
  </Card>
  <Card title="📈 Google Analytics" icon="graph">
    Ruch, zaangażowanie, konwersje do priorytetyzacji treści
  </Card>
</CardGrid>

Plugin działa w pełni bez żadnych podłączonych narzędzi — web search zapewnia dane zastępcze dla wszystkich komend. Podłączone narzędzia dodają precyzję.
