## What is SEO Machine?

SEO Machine is a plugin for Claude-powered tools that turns Claude into a complete SEO workstation. It provides 16 slash commands, 5 knowledge skills, and 4 autonomous agents covering the full SEO lifecycle — from keyword research through content creation, optimization, technical audits, and AI search visibility.

## Commands at a Glance

### Content Creation & Optimization

| Command | What it does |
|---------|-------------|
| `/research` | Keyword research, SERP analysis, competitive gap identification |
| `/write-seo` | Write SEO-optimized long-form content (2,000-3,000+ words) |
| `/optimize` | Analyze and optimize existing content with 0-100 scoring |
| `/analyze` | Deep SEO quality analysis with multi-module scoring |
| `/content-brief` | Generate writer-ready content briefs |
| `/meta-tags` | Generate optimized meta titles, descriptions, and OG tags |
| `/rewrite` | Refresh declining content to recover rankings |

### Site Architecture & Technical

| Command | What it does |
|---------|-------------|
| `/audit` | Full-site SEO audit — technical, on-page, content, competitive |
| `/topic-clusters` | Map topic clusters with pillar pages and internal linking |
| `/internal-links` | Strategic internal linking analysis and recommendations |
| `/schema-gen` | Generate JSON-LD schema markup for any page type |
| `/low-hanging-fruit` | Find quick-win keywords and pages for immediate gains |

### GEO & AI Search

| Command | What it does |
|---------|-------------|
| `/geo-audit` | Audit visibility in AI search engines (ChatGPT, Perplexity, Gemini) |
| `/llms-txt` | Generate an llms.txt file for AI crawler optimization |

### Local SEO

| Command | What it does |
|---------|-------------|
| `/local-seo` | Local SEO audit — GBP, citations, NAP, local keywords |

### Strategy

| Command | What it does |
|---------|-------------|
| `/competitors` | Deep competitive analysis with battlecards |

## Integrations

SEO Machine connects with external tools via MCP (Model Context Protocol) for enhanced data precision:

<CardGrid>
  <Card title="Ahrefs" icon="magnifier">
    Keyword volumes, difficulty scores, backlink data, ranking positions
  </Card>
  <Card title="Similarweb" icon="bars">
    Traffic estimates, audience data, competitive intelligence
  </Card>
  <Card title="Google Search Console" icon="document">
    Your actual impressions, clicks, CTR, and positions
  </Card>
  <Card title="Google Analytics" icon="graph">
    Traffic, engagement, conversions for content prioritization
  </Card>
</CardGrid>

The plugin works fully without any connected tools — web search provides fallback data for all commands. Connected tools add precision.
