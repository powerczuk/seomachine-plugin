# Programmatic SEO

Frameworks for scaling SEO content production, building topic authority through cluster architecture, and prioritizing content investments using data.

## Topic Cluster Architecture

### Structure: Hub-and-Spoke Model

Build topical authority through interconnected content networks:

**Pillar Page** (hub):
- Comprehensive 3,000-5,000+ word guide covering entire topic
- Targets broad, high-volume keyword (e.g., "email marketing")
- Links out to every cluster page
- Updated quarterly with new developments
- Table of contents with jump links

**Cluster Pages** (spokes):
- Focused 1,500-2,500 word articles on subtopics
- Target long-tail keywords (e.g., "email marketing subject line formulas")
- Link back to pillar page with descriptive anchor text
- Cross-link to 2-3 related cluster pages
- Published on a cadence (weekly/biweekly)

**Supporting Content**:
- FAQ pages targeting question-based queries
- Glossary pages for definition keywords
- Case studies providing E-E-A-T signals
- Tool/template pages for transactional intent

### Cluster Mapping Process

1. **Seed keyword identification** — start with broad topic keywords
2. **Subtopic discovery** — use "People Also Ask", related searches, keyword tools
3. **Intent mapping** — classify each subtopic by search intent
4. **Gap analysis** — identify subtopics not covered by competitors
5. **Prioritization** — score by search volume × relevance × difficulty
6. **Internal linking plan** — map connections between pillar and cluster pages
7. **Content calendar** — schedule publication for maximum cluster effect

## Data-Driven Content Prioritization

### Opportunity Scoring (8-Factor Model)

Score content opportunities on a 0-100 scale using these weighted factors:

| Factor | Weight | Data Source |
|--------|--------|-------------|
| Search volume | 20% | Keyword tools / GSC |
| Keyword difficulty | 15% | Keyword tools |
| Current ranking position | 15% | GSC |
| Content gap (vs. competitors) | 15% | SERP analysis |
| Business relevance | 15% | Manual assessment |
| Conversion potential | 10% | Intent + funnel stage |
| Content freshness need | 5% | Last update date |
| Trending momentum | 5% | Search trends |

### Quick Wins Identification

Find high-impact, low-effort opportunities:

- **Position 11-20 keywords** — almost on page 1, small improvements push them over
- **High-impression, low-CTR pages** — title/description optimization can double clicks
- **Missing meta descriptions** — quick fix with immediate CTR impact
- **Thin content on ranking pages** — expand word count on pages already getting traction
- **Internal link opportunities** — add links from high-authority pages to underperformers

### Declining Content Detection

Flag content losing organic performance:

- Pages with 20%+ traffic decline month-over-month
- Keywords dropping 5+ positions in 30 days
- Pages with decreasing impressions over 90 days
- Content older than 12 months without updates

**Refresh strategy**: Update statistics, add new sections, refresh examples, improve meta elements, strengthen internal links, add schema markup.

## Programmatic Content Templates

### Template-Based Page Generation

For businesses with repeatable content patterns, create templates:

**Location pages**: "[Service] in [City]" — template with dynamic city data, local schema, localized testimonials
**Comparison pages**: "[Product A] vs [Product B]" — structured comparison with feature matrix, pricing, verdict
**Integration pages**: "[Product] + [Integration]" — use case, setup guide, benefits
**Use case pages**: "[Product] for [Industry/Role]" — targeted messaging, relevant case studies

### Template Requirements
- Minimum 60% unique content per page (avoid thin/duplicate penalties)
- Unique meta titles and descriptions per page
- Localized/contextualized data points
- Unique images or screenshots where possible
- Internal links to related programmatic pages and pillar content

## Landing Page SEO + CRO

### Above-the-Fold Optimization (5-Second Test)

Visitors decide to stay or leave in 5 seconds. The fold must answer:
1. What is being offered?
2. Who is it for?
3. What should I do next?
4. Why should I trust this?

### CTA Optimization Framework

- **Formula**: Action verb + benefit word ("Start your free trial", "Get the guide")
- **Placement**: Primary CTA above fold, secondary after value sections, strong close CTA
- **Risk reversal**: Add trust signals near CTAs ("No credit card required", "Cancel anytime")
- **Contextual variation**: Different CTA copy at different scroll depths
- **A/B testing**: Always test CTA copy, color, placement, and size

### Trust Signal Architecture
- Testimonials with real names, titles, companies, and specific results
- Customer count / social proof metrics ("Trusted by 10,000+ teams")
- Media logos and "as seen in" sections
- Security badges near forms
- Risk reversal near CTAs

## Content Performance Analytics Integration

### Key Metrics to Track

**Search Console**:
- Impressions, clicks, CTR, average position by page and query
- Index coverage issues
- Core Web Vitals scores
- Mobile usability issues

**Analytics**:
- Organic sessions by landing page
- Bounce rate and engagement time by page
- Conversion rate by organic landing page
- Assisted conversions from organic content

### Reporting Cadence
- **Weekly**: Ranking movements, traffic trends, quick win identification
- **Monthly**: Full performance report, content gap refresh, competitor movement
- **Quarterly**: Topic cluster audit, pillar page refresh, strategy recalibration
