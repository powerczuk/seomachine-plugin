# /content-brief — SEO Content Brief Generator

Generate a detailed, writer-ready content brief for the target keyword. This brief provides everything a writer needs to create content that ranks.

Target: `$ARGUMENTS`

## Research Phase

Use web search to gather:

### Keyword Data
- Primary keyword and search intent
- 5-8 secondary keywords and variations
- 3-5 question-based keywords (People Also Ask)
- 3-5 LSI/semantic keywords

### Competitive Landscape
- Analyze top 5 SERP results for the primary keyword
- Note content types, lengths, and structures
- Identify must-cover topics (common across all top results)
- Find differentiation opportunities (gaps in existing content)
- Check for featured snippet opportunities

## Content Brief Output

### Brief Header
- **Target keyword**: [primary keyword]
- **Search intent**: [informational/commercial/transactional]
- **Target word count**: [based on competitive analysis, typically median + 20-30%]
- **Target audience**: [who is searching for this]
- **Content type**: [blog post/guide/comparison/landing page]
- **Difficulty assessment**: [easy/moderate/hard — based on competition]

### Keywords to Include
| Type | Keywords | Usage Guidance |
|------|----------|---------------|
| Primary | [keyword] | H1, first 100 words, 2-3 H2s, conclusion, 1-2% density |
| Secondary | [list] | Natural body copy, some H2/H3s |
| Questions | [list] | Use as H2 headings for snippet targeting |
| LSI/Semantic | [list] | Sprinkle naturally throughout |

### Recommended Outline

```
H1: [Suggested headline with primary keyword]

H2: [Section 1 — must-cover topic]
  H3: [Subsection if needed]
  H3: [Subsection if needed]

H2: [Section 2 — must-cover topic]

H2: [Section 3 — differentiator section]
  H3: [Subsection]

H2: [Section 4 — question-based (snippet target)]

H2: [Section 5 — differentiator/unique angle]

H2: [Section 6 — practical/actionable section]

H2: Conclusion / Key Takeaways
```

For each section, include:
- **What to cover**: 2-3 sentence description of the section's purpose
- **Key points to hit**: Specific facts, data, or arguments to include
- **Competitor coverage**: How top results handle this topic
- **Differentiation angle**: How to do it better

### Meta Elements (Draft)
- **Title tag options** (3): 50-60 chars each, keyword included
- **Meta description options** (3): 150-160 chars, keyword + value prop + CTA
- **URL slug**: Clean, keyword-included, 3-5 words

### Featured Snippet Strategy
- Target format: [paragraph/list/table/definition]
- Target query: [the specific question to answer]
- Placement: Immediately after relevant H2
- Format requirements: [40-60 words for paragraph, 5-9 items for list, etc.]

### Internal Linking Targets
Suggest 3-5 existing pages/topics that should be linked from this content.

### External Source Suggestions
Suggest 2-3 authoritative external sources to reference for credibility.

### Schema Markup
Recommended schema type and key fields to implement.

### Engagement Requirements
- Hook type recommendation (with example)
- Mini-story placement suggestions
- CTA placement and type suggestions
- Visual content notes (where images/charts/tables add value)

### Quality Standards
- Minimum word count: [target]
- Readability: Flesch 60-70, Grade 8-10
- Keyword density: 1-2% primary
- Internal links: 3-5+
- External links: 2-3
- Heading frequency: Every 300-400 words
- Publishing score threshold: 70/100
